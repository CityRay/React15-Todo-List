# React 15 Todo List Sample

This is a simple todo list app built with React, Redux, Webpack and Eslint.

- React v15.0.2
- Redux v3.5.2
- Eslint v2.9.0

### Install

```javascript
npm install
```

### Run Service & Develop

```javascript
npm run serve
```

### Lint React

```javascript
npm run lint
```
